# 📋 RFID Attendance System with Google Sheets

## Project Overview

The RFID Attendance System is an IoT-based smart attendance management solution that automates attendance tracking using RFID technology and cloud-based data storage. The system records user attendance by scanning RFID cards and automatically updates attendance records in Google Sheets for real-time monitoring and management.

This project eliminates manual attendance processes, improves accuracy, and provides a centralized platform for attendance management.

## System Architecture

The RFID reader scans authorized RFID cards and sends the data to the microcontroller. The attendance information is processed and transmitted to Google Sheets through an internet connection. The system maintains real-time attendance records that can be accessed remotely.
